Simplify object with given method.

Meshes.SimplificationMethod — Type
SimplificationMethod

A method for simplifying geometric objects.

source

Douglas-Peucker

Meshes.DouglasPeucker — Type
DouglasPeucker(ϵ)

The Douglas-Peucker algorithm for simplifying polygonal chains or polygonal areas given a deviation tolerance ϵ.

References

  • Douglas, D. and Peucker, T. 1973. Algorithms for the Reduction of the Number of Points Required to Represent a Digitized Line or its Caricature
